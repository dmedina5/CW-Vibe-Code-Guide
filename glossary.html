📖 Glossary

A comprehensive guide to AI and vibe coding terminology. All terms explained in plain, non-technical language.

💡 How to Use This Glossary

Terms are organized into categories. If you encounter an unfamiliar word anywhere in this guide, you can find its explanation here. Use Ctrl+F (or Cmd+F on Mac) to search for specific terms.

Core AI Concepts

Fundamental terms for understanding how AI assistants work.

Term What It Means
Vibe Coding A way of working where you describe what you want in plain English and an AI assistant helps create it. You guide the direction while the AI does the technical work. Think of it as having a conversation with a smart assistant who can actually do things for you.
AI / Artificial Intelligence Computer software designed to perform tasks that typically require human intelligence, like understanding language, recognizing patterns, and making decisions. Claude and Gemini are AI systems.
LLM (Large Language Model) The type of AI that powers tools like Claude and Gemini. These models are trained on vast amounts of text to understand and generate human-like responses. Think of it as the "brain" behind AI assistants.
Model A specific version of an AI system. For example, "Claude Opus 4.5" and "Claude Sonnet 4.5" are different models from Anthropic, each with different capabilities. Newer models are generally smarter and more capable.
Agentic AI AI that can take actions on its own (like reading files, searching, or making changes) rather than just answering questions. It's like an assistant who can actually do tasks, not just give advice. Both Claude Code and Gemini CLI are agentic.
Prompt The instructions or questions you give to an AI. A good prompt clearly explains what you want the AI to do. Writing effective prompts is sometimes called "prompt engineering."
Context The information the AI knows about when answering your question. This includes your current conversation, files you've shared, and previous instructions. More context usually leads to better, more relevant answers.
Tokens The way AI measures text. A token is roughly 3-4 characters or about 3/4 of a word. "Hello world" is about 2-3 tokens. AI models have limits on how many tokens they can process at once (context window) and how many they can generate (output limit).
Context Window The maximum amount of information (measured in tokens) an AI can "see" at once. A larger context window means the AI can consider more files and conversation history when responding. Gemini 3 Pro has a 2 million token window - enough to hold entire codebases.
Hallucination When an AI generates information that sounds plausible but is incorrect or made up. This can happen when the AI doesn't have enough information or is asked about topics outside its training. Always verify important facts.
Temperature A setting that controls how creative or predictable the AI's responses are. Lower temperature = more focused and consistent. Higher temperature = more creative and varied. Most AI tools set this automatically.

Tools & Products

The specific AI tools and applications referenced in this guide.

Term What It Means
Claude Code An AI assistant made by Anthropic that can help with coding, writing, analysis, and many other tasks. Available as a Desktop App (easy) or Terminal tool (for developers). It can read and edit files, run commands, and work autonomously on tasks.
Gemini CLI Google's AI assistant that runs in the terminal. Known for its large context window (can hold entire codebases) and generous free tier. Good for research, large-scale analysis, and batch operations.
Claude Desktop App A regular application you download and install on your computer (like Word or Chrome). The Claude Desktop App lets you use Claude Code without needing to type terminal commands - just a visual chat interface.
Anthropic The company that makes Claude. Founded by former OpenAI researchers, Anthropic focuses on AI safety and building helpful, harmless, and honest AI systems.
Google Antigravity Google's AI-first IDE (code editor) announced in late 2025. Free for individual use, it features native Gemini 3 integration and multi-agent support for orchestrating multiple AI assistants in parallel.

Features & Modes

Specific capabilities and operating modes of AI tools.

Term What It Means
Plan Mode A mode in Claude Code where the AI creates a detailed plan before taking any actions. Useful for complex tasks where you want to review the approach before execution. Toggle with Shift+Tab.
Extended Thinking A feature that lets the AI "think longer" about complex problems before responding. Results in more thorough, well-reasoned answers but takes more time. Toggle with Alt+T (Windows) or Option+T (Mac).
Deep Think Gemini's version of extended thinking. A model variant (gemini-3-deep-think) optimized for complex reasoning and planning tasks that require careful analysis.
Sandbox Mode A safety feature that restricts what the AI can do on your system. When enabled, the AI can only make changes in isolated/safe ways. Useful when you want to be extra careful about what actions the AI takes.
Headless Mode Running an AI tool without an interactive interface - useful for automation. For example: claude "analyze this file" runs Claude, gets an answer, and exits. Great for scripts and automated workflows.
YOLO Mode A setting (mostly in Gemini CLI) that auto-approves AI actions without asking for confirmation. Speeds up workflows but reduces your control. Use carefully. Toggle with Ctrl+Y.
Permissions Settings that control what actions the AI is allowed to take. Can be set to require approval for each action, auto-approve safe actions, or auto-approve everything. More restrictive = safer but slower.
Multi-Agent Using multiple AI assistants at the same time, each handling different parts of a task. For example, using Gemini for research while Claude writes code. Can significantly speed up complex projects.

Configuration & Files

Files and settings used to customize AI tool behavior.

Term What It Means
CLAUDE.md A configuration file you create in your project folder to give Claude Code persistent instructions. Claude reads this file at the start of every conversation, so it "remembers" your preferences and project context.
GEMINI.md The equivalent configuration file for Gemini CLI. Contains project-specific instructions, coding standards, and context that Gemini should always know about.
Skills (Claude Code) Custom commands you can create for Claude Code. A skill is a reusable set of instructions stored in a file. For example, a "/search" skill could define exactly how Claude should search your knowledge base.
MCP (Model Context Protocol) A way to extend AI tools with additional capabilities by connecting them to external services. For example, an MCP server could let Claude control a web browser or access a database.
Markdown Files (.md) Simple text files that use special symbols (like # for headings or * for bullets) to create formatted documents. CLAUDE.md and GEMINI.md are markdown files. Easy to read and write.
API Key A password-like code that lets a program access a service. It's how AI tools verify you have permission to use them. Keep API keys secret - don't share them or commit them to code repositories.
Environment Variable A setting stored in your computer's system that programs can read. API keys are often stored as environment variables (like ANTHROPIC_API_KEY) so they're not visible in your code.

Technical Terms

Computer and software terms you may encounter.

Term What It Means
Terminal / Command Line A text-based way to interact with your computer by typing commands. It looks like a black window with text. Both Claude Code and Gemini CLI run in the terminal. Also called "shell" or "console."
CLI (Command Line Interface) A program you use by typing text commands instead of clicking buttons. "Gemini CLI" means Gemini that you control by typing commands in a terminal.
IDE (Integrated Development Environment) A software application where developers write and edit code. Think of it like Microsoft Word, but for writing computer programs. Examples: VS Code, Cursor, Google Antigravity.
npm (Node Package Manager) A tool for downloading and installing programs made with JavaScript. Think of it like an app store for developer tools. Used to install Claude Code and Gemini CLI.
PATH A list of folders where your computer looks for programs. When you type a command like "claude", your computer searches these folders to find the program to run. If a command "isn't found," it's often a PATH issue.
Git / Git Bash Git is a tool developers use to track changes to files and collaborate. Git Bash is a terminal program that lets you use Git (and other Unix-style commands) on Windows. Required for Claude Code on Windows.
Repository (Repo) A folder containing a project's files along with the history of all changes made to them. GitHub hosts repositories online so teams can collaborate.
Script A file containing a sequence of commands that run automatically. Scripts can automate repetitive tasks - like the workflow automations described in this guide.
Python A popular programming language known for being easy to read and learn. Often used for data analysis, automation, and AI applications. Many workflow automations in this guide use Python scripts.
Authentication The process of proving who you are (usually by logging in). Required before you can use Claude Code or Gemini CLI. May involve signing into your Anthropic or Google account.

Common Commands

Frequently used commands in Claude Code and Gemini CLI.

Command What It Does
/add Adds a file to Claude's context so it can read and reference it. Example: /add report.xlsx lets Claude analyze that spreadsheet.
/clear Clears the current conversation and starts fresh. Useful when you want to change topics or the AI seems confused.
/compact Summarizes the conversation to reduce token usage while keeping important context. Use when conversations get long.
/help Shows available commands and how to use them. Good starting point when you're not sure what you can do.
@filename In Gemini CLI, references a file. Example: @data.csv "summarize this" tells Gemini to read and summarize that file.
# In Claude Code, pressing # lets you quickly add a note to your CLAUDE.md file without leaving the conversation.
